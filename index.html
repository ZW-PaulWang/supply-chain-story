The Problem

Fragmented Operational Data

The simulated supply chain includes four sequential nodes: manufacturer, supplier, seller, and hospital[cite: 15].

When visibility is limited across these stages, minor changes in hospital demand create the bullwhip effect—causing firms to overproduce or suffer stockouts.

View 1

Executive Overview

The dashboard is organized into three analytical views[cite: 36]. This first view establishes immediate context.

By tracking physical flow and manufacturer margins, it calculates the current Quarter Gross Profit, answering immediately if the system is operating efficiently.

View 2

Flow & Capital Analysis

A Sankey diagram maps device movement. Crucially, users can toggle between Unit Volume mode and Capital mode.

Switching to Capital mode instantly highlights financial exposure, showing exactly how much capital is tied up in stagnant stock.

View 3

Forecasting True Demand

The third view introduces time series forecasting based on actual hospital consumption.

Using an ARIMA model, the dashboard projects next quarter's true hospital demand, cutting through the noise of intermediary seller orders.

Optimization

Predictive Profitability

Instead of a flat estimate, the dashboard models a dynamic profitability curve. Forecasted demand (light teal) is adjusted against existing downstream inventory.

The resulting Required Production (dark blue area) visually demonstrates how end-to-end visibility prevents overproduction and tightly protects margins.

Deep Dive

Granular Inventory Inspection

High-level metrics aren't always enough. The dashboard allows users to click directly into the supplier and seller nodes.

This reveals exactly which specific product types are currently held up in transit, granting immediate tactical visibility to operational teams.

Data Access

Export & Audit

Transparency builds trust. Users aren't locked into the visualizations.

The dedicated Data Hub allows users to inspect the exact synthetic datasets powering their current filters and securely download the clean CSV data for external auditing.
